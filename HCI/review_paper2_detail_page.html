From Gap to Synergy: Enhancing Contextual Understanding through Human-Machine Collaboration in Personalized System

Authors: Chen, Weihao, et al.

Conference: Proceedings of the 36th Annual ACM Symposium on User Interface Software and Technology, 2023.

Brief Introduction: 사용자가 어떤 환경, 행위를 하고 있는지 자동으로 파악하고, 동시에 사용자가 자신의 상태를 최종 컨펌하여 보다 개인화된 시스템을 만드는 인간과 기계의 협업 프로세스를 개발한 연구


Figure 1: 논문 대표 이미지


이 논문은 시스템 개인화를 위해 인간과 기계가 협업하여 사용자의 문맥 (의도, 환경 등)이해 능력을 강화하는 방법을 연구했습니다. AI + IoT 기술이 발전하고, 스마트폰 센서를 통한 정보 수집이 가능해졌으며 데이터 기반 기계학습 알고리즘이 발달되었기 때문에 자연스레 시스템 개인화에 대한 니즈가 생겼습니다. 그렇기에 어떤 방식으로 사용자 맞춤형 시스템을 만들지에 대한 연구가 필요하죠.


이 논문은 AI와 IoT 기술의 발전으로 지능형 시스템의 컨텍스트 인식 능력이 강화되고 있으나, 개별 사용자의 컨텍스트를 정확하게 이해하고 맞춤화하는 데는 여전히 한계가 있다는 문제의식에서 출발합니다. 사용자가 자신만의 IF-THEN 규칙을 맞춤 설정할 수 있게 하는 Trigger Action Programming (TAP)과 같은 패러다임에도 불구하고, 사용자들은 장치의 인지 능력에 대한 지식 없이는 개인화된 컨텍스트를 구축하는 데 어려움을 겪습니다. 이러한 컨텍스트 구축에서의 인간과 기계 간의 이해 차이를 극복하기 위해, 본 연구에서는 LangAware라는 시스템을 제안했습니다.

Context-aware computing

Context-aware computing이란, 사용자의 컨텍스트를 기반으로 맞춤형 서비스를 제공해주는 기술을 말합니다. 여기서 컨텍스트란 사용자의 환경, 활동, 선호도 등을 뜻합니다. 스마트폰이나 스마트 워치에서 시간, 위치, 활동, 앱 사용 기록, 근처 Wi-Fi/Bluetooth 기기 등의 물리적 감지 정보를 얻을 수 있으니 이를 이용해서 사용자의 컨텍스트를 추론할 수 있는 것이죠. 사용자의 풍부한 컨텍스트 정보는 추천 시스템, 메시지 관리, 필요한 앱 추천, 배터리 관리, 볼륨/밝기 관리 등 다양한 개인화된 자동 서비스를 제공할 수 있게 해주는 중요한 기술입니다. 주로, 센서로부터 얻어온 정보들을 머신러닝을 사용해서 인식해 사용하고 있습니다.

  • 컨텍스트 정보 수집 센서, 사용자 입력, 네트워크 정보 등 다양한 출처에서 데이터를 수집합니다. 이 데이터에는 사용자의 위치, 활동 상태, 시간, 주변 환경 등이 포함될 수 있습니다.
  • 컨텍스트 인식 수집된 데이터를 바탕으로 시스템은 현재의 사용자 상황을 인식합니다. 이 과정에서 데이터는 처리되고 해석되어 사용자의 현재 상태와 필요를 반영하는 컨텍스트 정보로 변환됩니다.
  • 적응적 응답 시스템은 인식된 컨텍스트를 바탕으로 사용자의 필요와 상황에 맞는 적응적인 응답을 제공합니다. 예를 들어, 사용자의 위치 정보를 바탕으로 가까운 식당을 추천하거나, 사용자의 일정에 따라 알림을 조정하는 것 등이 있습니다.

    • 따라서 스마트홈, 헬스케어, 모바일 서비스, 스마트 시티 등 다양한 응용 분야에 적용 가능합니다.


    하지만 아직 여러 개선 가능한 문제들이 있습니다.
  • 몇 달의 걸친 기간 동안 수집된 수많은 양의 사용자 데이터가 필요하고,
  • 맞춤형 컨텍스트 및 행동 데이터가 희소할 때에는 개인화된 지식을 얻는 데 비효율적이며,
  • 사용자에게 불투명한 AI로 작동하여 시스템의 행동을 이해하거나 제어하는 데에 어려움이 있습니다.
    • 그렇기 때문에, 기기가 스스로 처음부터 끝까지 알아서 진행하고, 사용자의 컨텍스트를 추론하는 것보다 사용자를 적절히 컨텍스트 분석에 참여시키는 것이 중요합니다. 기기가 추론한 결과를 사용자가 컨펌해준다면 더욱 정확한 개인화가 가능하겠죠! 그럼 기기의 추론과 사람의 컨펌을 언제, 어떤 방식으로 진행하면 좋을까요?

    Main Idea: LangAware

    Figure 1: 논문 대표 이미지



    본 논문에서 제시한 LangAware 접근법은 사용자가 자연어를 사용하여 현장에서 컨텍스트 규칙을 설정할 수 있게 하는 협력적 접근 방식입니다. 이 시스템은 LLM(Large Language Models)을 활용하여 사용자가 표현한 고수준의 컨텍스트 개념을 기계가 인지하고 구현할 수 있는 감지 능력과 의미적 연결을 합니다. 본 논문의 주요 개념인 Shared Contextual Concepts(SCC)는 인간과 기계 간의 컨텍스트 이해의 조율을 위한 매체로 제안되며, SCC는 자연어 구문과 해당하는 기계 부울 표현의 쌍으로 정의됩니다.

    Results

    LangAware를 통한 사용자 연구에서는 16명의 참가자가 실제 생활 환경에서 12개의 캠퍼스 시나리오를 통해 평균 87.5%의 성공률로 컨텍스트 규칙을 정의했습니다. LangAware는 기준 시스템 대비 성공률, 사용자 만족도 및 기타 측면에서 우수한 성능을 보여주며, 현장 컨텍스트 데이터를 통합함으로써 컨텍스트 협업의 효과가 있었다고 합니다.

    Implications

    LangAware와 같은 혁신적인 접근 방식은 사용자가 자신의 컨텍스트를 보다 정확하게 정의하고 맞춤 설정할 수 있게 함으로써, 인간-기계 상호작용의 직관성과 적응성을 향상시키고 IoT 시나리오에서 더 많은 맞춤형 컨텍스트 인식 응용 프로그램의 개발을 용이하게 합니다

    Future Work

  • 사용자의 궁극적인 목표 이해하기
    • 사용자가 설정한 규칙이나 명령어 뒤에 숨겨진 궁극적인 목표나 의도를 파악할 수 있는 알고리즘 개발
    • 사용자의 행동 패턴, 과거의 컨텍스트 규칙 설정 기록, 일정 등을 분석하여 사용자의 실질적인 요구 사항을 추론
  • 사용자 행동 패턴 마이닝을 통한 더 치밀한 개인화
    • 장기적인 운영을 위해 데이터 기반의 사용자 행동 패턴 마이닝 방법 연구.
    • 사용자의 일상적인 활동과 컨텍스트 규칙 설정 패턴을 분석하여 맞춤형 권장 사항 제공.
  • 협력을 이끄는, 효율적인 디바이스 개발
    • 사용자가 쉽게 기기의 추론에 피드백할 수 있게 하는 새로운 디바이스, 인터랙션 개발 (햅틱, 음성 등)

    User는 때때로 Chatbot과의 최적의 협력 방법을 찾기 위해 시행착오를 거쳐야할 수도 있습니다. 따라서 User에게 에이전트가 필요한 동작을 구체화하는 동사를 말하게 하던가, Chatbot이 어떻게 작동되는지, 특정 용어가 뭔지, 어떠한 표현을 이해하지 못했는지, 언어 처리에 대한 한계는 어떠 한지를 상황에 맞게 더 많이 설명하면 User는 최적의 전략을 찾기 수월해질 것입니다.
